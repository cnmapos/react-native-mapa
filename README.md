# 文档生成
`typedoc`API文档生成使用`typedoc`，生成结果为静态html文件，可直接执行。如果为组件添加注释，可查看[官网](https://typedoc.org/guides/overview/)。

执行命令`yarn run doc:gen`生成API文档。执行命令`yarn run doc:start`启动本地服务查看生成的API文档。
